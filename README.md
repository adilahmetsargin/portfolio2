# Solution Portfolio

A single portfolio hub for software projects that combine business thinking, interactive UI, data visibility, and practical workflow design.

The site is positioned around a simple idea: not just building React screens, but building software that helps people understand work, make decisions, and move faster.

## What It Showcases

- **Automotive and 3D demos**: interactive Three.js experiences for manufacturing and vehicle quality use cases.
- **Internal tools**: operational dashboards, supplier follow-up workflows, and shift handoff concepts.
- **Data applications**: KPI views, risk signals, filtered queues, and decision-support interfaces.
- **Web applications**: polished product-style UI for client intake and business workflows.

## Featured Projects

### 3D Assembly Line Digital Twin

A browser-based 3D production line monitor for supervisors and manufacturing software teams.

It demonstrates:

- Station health
- Takt pressure
- Work-in-progress flow
- Bottleneck and downtime risk
- OEE-style operating signals

### Vehicle Quality Intelligence

A quality investigation console for warranty signals, defect clusters, severity, and likely root-cause areas.

It demonstrates:

- 3D vehicle inspection UI
- Defect heatmap thinking
- Claim severity triage
- Root-cause action guidance

### Operations Command Center

A practical KPI dashboard concept for daily business operations.

It demonstrates:

- Throughput and downtime metrics
- Open issue tracking
- Team coverage signals
- Shift or daily mode switching

### Supplier Follow-up CRM

A lightweight workflow tool for vendor follow-up and delivery risk tracking.

It demonstrates:

- Supplier status queues
- Risk filtering
- Owner visibility
- Follow-up workflow design

### Shift Handoff Board

A focused communication tool for passing open issues between teams or shifts.

It demonstrates:

- Status filtering
- Owner assignment
- Operational notes
- Simple workflow clarity

## Local Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Project Structure

```text

  main.jsx      # React app, project data, interactions, and Three.js scenes
  styles.css    # Layout, responsive styles, and visual system
```
